## dotfiles galore

These are configurations of my system, just they way i like it.

Feel free to clone/fork.

## Installation.

Bootstrap dotfiles

```
  $ git clone https://github.com/lestoni/dotfiles.git
  $ ./dotfiles/bootstrap.sh
```

Install packages

```
  $ sudo ./bin/install
```

Install Vim plugins

```
  $ cd ~/.vim
  $ ./install.sh
```
